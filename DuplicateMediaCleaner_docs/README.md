# DuplicateMediaCleaner プロジェクト資料

Windows専用の類似・重複画像/動画削除アプリケーションの設計資料一式です。

## 📁 ファイル構成

| ファイル名 | 内容 |
|-----------|------|
| **01_要件定義書.md** | 機能要件・非機能要件・UI仕様・実装ロードマップ |
| **02_技術スタック決定書.md** | 採用技術・アーキテクチャ・API設計・開発計画 |
| **03_UIワイヤーフレーム.html** | インタラクティブなUIモックアップ（ブラウザで開く） |

## 🎯 プロジェクト概要

### 目的
Windowsユーザーが、ローカルストレージに保存された画像・動画ファイルから、重複ファイルおよび類似ファイルを検出し、安全に削除できるデスクトップアプリケーション。

### 主要機能
- ✅ 画像・動画の独立検出
- ✅ 完全重複検出（ハッシュベース）
- ✅ 類似ファイル検出（知覚ハッシュ）
- ✅ 類似度スレッショルド調整（0-100%）
- ✅ カテゴリ別の削除確認
- ✅ 動画は先頭フレームでプレビュー
- ✅ 画像の横並び比較

## 🛠️ 技術スタック

### フロントエンド
- Electron 28.x
- React 18.x + TypeScript 5.x
- Tailwind CSS + shadcn/ui
- Zustand（状態管理）

### バックエンド
- Python 3.11+
- FastAPI + WebSocket
- OpenCV + Pillow + imagehash（画像処理）
- ffmpeg-python（動画処理）
- SQLite（キャッシュ）
